Геральт получил от заказчика сложное задание, для выполнения которого потребуется применить все свои навыки. Заказчик хочет проверить, насколько хорошо ведьмак умеет справляться с трудными задачами. Он предлагает Геральту определить есть ли среди трёхзначного числа одинаковые цифры.

Входные данные

Вводится неотрицательное трехзначное целое число.

Выходные данные

Выведите YES если число содержит одинаковые цифры и NO в противном случае.

Примеры

Входные данные
122
Выходные данные
Yes
Входные данные
123
Выходные данные
No
