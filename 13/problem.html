Во время своего путешествия Геральт сталкивается с различными существами, некоторые из которых могут быть опасны для него. Чтобы победить этих существ, Геральт использует свои магические способности и оружие. Однако, чтобы использовать магию, Геральту нужны специальные руны, которые можно найти в различных местах мира The Witcher. Ваша задача - написать программу нахождения для заданного натурального числа N всех делителей, кратных натуральным числам C и D одновременно.

Входные данные

Водятся числа N, C, D.

Выходные данные

Выведите искомые делители.
